# CSS FLEXBOX
## Descrição:
Repositório do curso de css flexbox, o diretório FLEX-BOX é um pouco sobre como funciona o display flex e algumas opções de posicionamento e o diretório app foi um clone de uma página do twitter utilizando somente HTML e CSS, com todo o conteúdo estático somente para práticar o flexbox mesmo e para descontrair utilizei personagens de um desenho que gosto muito.
